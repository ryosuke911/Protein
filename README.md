# たんぱく図鑑

自分が食べた物のタンパク質を記録し、自分だけのタンパク質図鑑を作成することができるアプリ。

URL: http://43.206.18.203/

# 使用技術

• Ruby 2.7.6

• Ruby on Rails 6.1.7

• MySQL 8.0.31

• NginX

• Puma

• AWS

　　￮ VPC
  
　　￮ EC2
  
　　￮ RDS
  
# 機能一覧

• ユーザー登録、ログイン機能(devise)： 新規登録・ログイン・ログアウトが可能

• 投稿機能： 自分が食べた物のタンパク質量・価格・コメントの投稿が可能

• 投稿編集,削除機能

• 投稿一覧機能： 自分が投稿したデータのみ閲覧可能（＝自分の投稿が他人に閲覧されることはない）。
